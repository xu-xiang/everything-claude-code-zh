# Debugging 提问模板

## 1. 通用 Bug 分析
这是一个 Android 问题。请先不要直接改代码。
请按以下结构输出：
1. 问题理解
2. 可能根因
3. 最可能原因
4. 涉及文件
5. 最小修改方案
6. 验证步骤

## 2. Crash 分析
这是 crash 堆栈和相关代码。请先分析根因，再给出最小修复方案。
请说明：
- 是否为空指针 / 生命周期 / 并发 / 输入问题
- 是否可能引入回归
- 如何验证修复

## 3. ANR / 卡顿分析
请基于 `contexts/performance-context.md`，
分析这个 ANR / 卡顿问题的可能原因，并按概率排序。
请给出最小优化点、风险和验证指标。
