# 变更审查检查清单

在接受 Copilot 生成的代码前，请人工检查：

## 行为
- 是否改变了现有外部行为？
- 是否影响兼容性？
- 是否引入额外副作用？

## 范围
- 改动是否最小？
- 是否修改了无关文件？
- 是否引入了不必要抽象？

## 可靠性
- 异常处理是否明确？
- 空值处理是否安全？
- 并发 / 生命周期是否正确？

## Android 特性
- 是否影响权限、生命周期、线程、状态恢复？
- 是否影响 Binder / AIDL / system service 稳定性？
- 是否引入 UI 状态错乱或重组问题？

## 可验证性
- 是否提供了测试建议？
- 是否给出了手动验证步骤？
