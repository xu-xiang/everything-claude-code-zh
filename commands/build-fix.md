# 构建与修复 (Build and Fix)

增量式修复 TypeScript 与构建错误：

1. 运行构建：npm run build 或 pnpm build

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 针对每个错误：
   - 显示错误上下文（前后 5 行）
   - 解释问题所在
   - 提出修复方案
   - 执行修复
   - 重新运行构建
   - 验证错误是否已解决

4. 停止条件：
   - 修复引入了新错误
   - 同一错误在 3 次尝试后依然存在
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误
   - 剩余的错误
   - 新引入的错误

为了安全起见，一次只修复一个错误！
